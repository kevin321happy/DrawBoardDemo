# DrawBoardDemo
一个画板的控件


两个自定义控件,一个画布的View，一个控制绘制操作的弹窗菜单

状态 | 功能
-------- | ---
**支持**|**曲线、矩形、椭圆等随手势的滑动进行绘制**
**支持**|**绘制图形的大小任意,可多种图案同时存在在画板上**
**支持**|**绘制的撤销、清屏操作**
**支持**|**绘制的画笔的颜色粗细的设置**
**支持**|**在此基础上拓展增加其他类型的图案的绘制**
